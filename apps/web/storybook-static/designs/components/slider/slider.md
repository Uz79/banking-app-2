# Slider

Slider input for selecting a value from a bounded range.

## Variants

- `default-default`

## States

- `default`
- `hover`
- `pressed`
- `focus`
- `disabled`

## Accessibility

- Expose an accessible name.
- Expose min, max, and current value.
- Support keyboard focus and arrow-key changes in implementation.
