# Card components

Grouped card patterns under `designs/components/card/`:

| Folder | Manifest | Role |
|--------|----------|------|
| `section-card/` | `section-card.json` | Dashboard section cards (accounts, payments, bookings, …) |
| `color-theme/` | `color-theme.json` | Profile saved / add theme cards |

Each subfolder follows the shared convention: `<slug>.json`, `<slug>.md`, optional `overview.png`, and `variants/<variant-kebab>/default.svg` (+ `default.png` when exported).
