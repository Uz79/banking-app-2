# Toggle switch

A selection control for enabling or disabling a setting.

## Variants

- `regular-active`: on
- `regular-inactive`: off

## States

- `default`: resting
- `hover`: pointer hover (web)
- `focus`: pressed/active feedback (exported as `Pressed` in Figma)

## Usage notes

- Use for immediate effect settings (on/off), not for multi-step forms.
- Labels should describe the state (“Notifications”) rather than the action (“Enable notifications”).
