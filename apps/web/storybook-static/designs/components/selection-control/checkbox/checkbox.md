# Checkbox

A selection control that allows users to toggle an independent boolean choice.

## Variants

- `regular-active`: checked
- `regular-inactive`: unchecked

## States

- `default`: resting
- `hover`: pointer hover (web)
- `focus`: pressed/active feedback (exported as `Pressed` in Figma)

## Usage notes

- Use when multiple options can be selected independently.
- Use a clear label and keep the target area large enough for touch.
