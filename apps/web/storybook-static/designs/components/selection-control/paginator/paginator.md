# Paginator

A compact page/step indicator used to show position in a sequence (e.g. carousel, onboarding, or multi-step flows).

## Variants

- `3-dots-left-active`: left dot active
- `3-dots-mid-active`: middle dot active
- `3-dots-right-active`: right dot active

## States

- `default`: exported snapshots for each active position (no hover/pressed exports yet)

## Usage notes

- Use for short sequences; keep dot count small to avoid ambiguity.
- Pair with swipe/next/previous controls when pagination is interactive.
