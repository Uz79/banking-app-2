## Expander

Expandable row header (accordion trigger) with a leading icon container and a chevron.

### Variants

- `regular`

### States

- `default`
- `hover`
- `pressed`

### Notes

- The leading icon container uses a primary tint layer: 5% default → 10% hover → 20% pressed.
- The row itself is intended to be a `<button>` for accessibility.
