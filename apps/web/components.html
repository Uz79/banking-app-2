Button tokens from tokens4_website-UZ.json (mapped/light and mapped/dark → css/tokens.css). Primary and tonal use solid surfaces for hover and pressed; secondary uses a 10% / 20% foreground tint on the fill (same layered intent as Figma “state layer”). Neutral rows (product-item, list-item, booking-row) use a ::before state overlay on a stable container (--uz-surface-bg); optional utility class uz-interactive-surface.
