# banking-app
AI-assisted banking UX prototype.

## Structure

- `apps/`
  - `web/` — web app
  - `ios/` — iOS app
  - `backend/` — API/services (optional)
- `packages/`
  - `ui/` — shared UI primitives
  - `tokens/` — shared design tokens
  - `shared/` — shared utilities/types
- `designs/` — design artifacts (components; **`screens/`** — same manifest shape as components: **`<slug>.json`**, **`<slug>.md`**, **`overview.png`**, **`variants/`**, see **`designs/screens/README.md`**; tokens)
- `docs/` — documentation
- `prompts/` — prompt library / agent workflows
