# Tab control

Tab control for switching between related content panels, such as RGB and HSL colour inputs.

## Variants

- `default-default`

## States

- `default`
- `hover`
- `pressed`
- `focus`

## Accessibility

- Use `role="tablist"` on the group.
- Use `role="tab"` on each option.
- Support keyboard focus and arrow-key navigation in implementation.
