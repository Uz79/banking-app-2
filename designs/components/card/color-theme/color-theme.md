# Color theme cards

Cards for adding, displaying, selecting, and deleting custom color themes (foreground/background pairs).

Sibling under `card/`: **`section-card/`** (dashboard section cards on Overview / Payments / Account details).

## Variants

- `add-theme`
- `saved-custom-theme`

## States

- `default`
- `selected` (for the saved theme card)

## Assets

- `variants/add-theme/default.svg`
- `variants/saved-custom-theme/default.svg`
- `variants/saved-custom-theme/selected.svg`
